Choisissez votre plateforme préférée pour déployer votre site en quelques clics!

---

## ⚡ OPTION 1: GITHUB PAGES (Recommandé - Gratuit)

### ✅ Étapes (30 secondes):

1. **Allez ici:** https://github.com/rezofabrik-hub/comte-guifr-/settings/pages

2. **Configurez:**
   - Source: `Deploy from a branch`
   - Branch: `claude/lettrage-configurator-site-t9zy3x`
   - Folder: `/(root)`
   - Cliquez **Save**

3. ✅ **Votre site sera live à:**
   ```
   https://rezofabrik-hub.github.io/comte-guifr-/
   ```

---

## ⚡ OPTION 2: NETLIFY (Très Simple - Gratuit)

### 🔗 Déploiement en 1 clic:

```
https://app.netlify.com/start/deploy?repository=https://github.com/rezofabrik-hub/comte-guifr-
```

**Ou manuellement:**

1. Allez sur: https://netlify.com
2. Cliquez "Add new site" → "Import an existing project"
3. Sélectionnez GitHub → `rezofabrik-hub/comte-guifr-`
4. Branch: `claude/lettrage-configurator-site-t9zy3x`
5. Cliquez Deploy

✅ **Votre site sera live à:**
```
https://rezo-decoupe.netlify.app
(ou un nom auto-généré)
```

---

## ⚡ OPTION 3: VERCEL (Le Plus Rapide - Gratuit)

### 🔗 Déploiement en 1 clic:

```
https://vercel.com/new/clone?repository-url=https://github.com/rezofabrik-hub/comte-guifr-&project-name=rezo-decoupe
```

**Ou manuellement:**

1. Allez sur: https://vercel.com/new
2. Sélectionnez "Import Git Repository"
3. Paste: `https://github.com/rezofabrik-hub/comte-guifr-`
4. Framework: `Other` (static)
5. Branch: `claude/lettrage-configurator-site-t9zy3x`
6. Cliquez Deploy

✅ **Votre site sera live à:**
```
https://rezo-decoupe.vercel.app
```
